Moved to Ghost

mike

Dec 22, 2021 • 1 min read

I came across Ghost before, but more recently in my attempt to save money by getting rid of the "big name CMS".  I always thought Ghost was hard to get going and didn't want to pay for a blog on their site.  Today I found out that it is somewhat easy to get going.  Here are the steps I took, for others looking to use Ghost and long-term storage on the blog.  Caveat...I'm on a Mac so please take steps per your operating system.

  • Wget was needed to pull the latest version of Ghost .zip.

wget --no-check-certificate https://ghost.org/zip/ghost-latest.zip

  • Now unzip into a ghost folder or whatever folder you want to launch Ghost from.

unzip ghost-latest.zip

  • Install Ghost using npm.  You'll need to install node, per your operating system, if you haven't already and it includes npm.

npm install --production

  • You should be able to run Ghost if all went well with the previous steps.

npm start

  • You got Ghost!  Get to bloggin' and creating pages.

The admin page should be available at mikedevs.com/ghost
