Install Ghost locally

  • Wget was needed to pull the latest version of Ghost .zip.

  • Now unzip into a ghost folder or whatever folder you want to launch Ghost from.

  • Install Ghost using npm. Â You'll need to install node, per your operating system, if you haven't already and it includes npm.

npm start

  • Ghost should start up and you can access the site using the address below (remove /ghost).
